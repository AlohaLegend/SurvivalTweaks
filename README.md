# SurvivalTweaks

SurvivalTweaks provides small survival-server quality-of-life behavior:

- `/biome` shows the biome a player is standing in.
- Rare player-head drops can occur from eligible PvP kills.
- Player-head rolls include playtime, same-address, killer, victim, and pair cooldown checks to discourage farming.

## Commands

- `/biome`
- `/survivaltweaks reload`
- `/survivaltweaks status`

## Permissions

- `survivaltweaks.biome`
- `survivaltweaks.admin`
